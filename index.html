Semestrální projekt z předmětu B0B39PGR

Chill Room Island

Ilia Timofeev

Assignment

The assignment for this course was to build an interactive 3D scene using a variety of models.
We needed to write our own shaders and implement loading of materials and lighting.
Additionally, the scene had to support user interaction by clicking on objects—specifically cubes, cat model, sphere, fire.
Finally, the project required adding dynamic fog effects and collision detection.

Control

W/A/S/D: Move camera forward/left/back/right in free-flight mode
Space/C: Move camera up/down
Shift: Toggle speed up
Mouse Left-Click: Toggle mouselook (when in free-flight mode)
Mouse Right-Click: Perform object picking and selection
1/Down Arrow: Switch to the dynamic camera
2: Switch to the first static camera
3: Switch to the second static camera
4: Switch to the cat camera
Left/Right Arrow: Switch between static camera viewpoints
F: Toggle flashlight
Left Control + G: Toggle scene fog
V: Toggle gravity
R: Reload all shaders at runtime
Esc: Exit the application
